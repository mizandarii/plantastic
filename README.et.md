# Plantastic
![freddie-marriage-UcfKYTan-LU-unsplash](https://github.com/user-attachments/assets/eee07940-7fe3-458b-89a5-156a44e7bff1)
Plantastic on mobiilirakendus, mis aitab kasutajatel hallata oma toataimi. Rakendus pakub **kastmise meeldetuletusi**, **taimede hoolduslogisid**, **taimede entsüklopeediat** ja **kogukonna küsimusi ja vastuseid**.
## Funktsioonid

- **Minu Taimed:** Lisa oma taimed, jälgi kastmisaegu, märgi kastmine tehtuks.
- **Nutikaid Meeldetuletusi:** Saad meeldetuletusi kastmise kohta, võimalus edasi lükata või logida kastmist.
- **Taimede Entsüklopeedia:** Teave populaarsete taimede kohta, sh kastmisvajadus, valguse vajadus ja hooldusnipid.
- **Kogukonna Q&A:** Küsi küsimusi, jaga nippe ja vaata teiste kasutajate vastuseid.
- **KKK / Hooldusjuhend:** Kontrollitud nõuanded taimede hoolduseks.
- **Võimalus Töötada Offline:** Taimede andmed ja kastmiskavad töötavad võrguühenduseta.
